# ERP系统项目文档

## 1. 项目概述

本项目是一个企业资源规划（ERP）系统，主要用于管理企业的人力资源、骑手管理、工作流程、合同管理等业务。系统采用前后端分离架构，前端使用HTML、CSS和JavaScript，后端使用Flask框架和MySQL数据库。

## 2. 目录结构

```
kunze_erp_system/
├── backend/            # 后端代码
│   ├── config/         # 配置文件
│   ├── models/         # 数据模型
│   ├── routes/         # 路由
│   ├── uploads/        # 上传文件
│   ├── utils/          # 工具函数
│   ├── app.py          # 应用入口
│   └── database.sql    # 数据库SQL文件
├── frontend/           # 前端代码
│   ├── css/            # 样式文件
│   ├── js/             # JavaScript文件
│   └── *.html          # HTML页面
└── uploads/            # 上传文件目录
```

## 7. 技术栈

- **前端**：HTML5, CSS3, JavaScript, jQuery
- **后端**：Python, Flask, Flask-CORS
- **数据库**：MySQL
- **其他**：pymysql (MySQL连接库)

## 8. 系统部署

1. 安装依赖：`pip install -r requirements.txt`
2. 运行数据库初始化：`python backend/app.py`
3. 访问系统：http://localhost:5000

## 9. 注意事项

- 系统使用MySQL数据库，需要确保数据库服务正常运行
- 上传文件存储在`backend/uploads`目录，需要确保该目录存在且有写入权限
- 系统默认管理员账号：admin / admin123
